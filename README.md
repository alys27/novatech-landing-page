# NovaTech — Landing Page

Müasir, tək-səhifəlik (single-page) landing page. Rəqəmsal həllər təqdim edən bir şirkət üçün hazırlanmışdır.

🔗 **Demo:** https://alys27.github.io/novatech-landing-page/

## Xüsusiyyətlər

- Responsiv dizayn (mobil, planşet, desktop)
- Scroll zamanı aktivləşən animasiyalar (Intersection Observer)
- Bölmələrə bağlı, dinamik yaradılan "circuit board" fon effekti
- Hero bölməsində canvas əsaslı hissəcik (particle) animasiyası
- Formanın client-side validasiyası (ad, email, mesaj sahələri)
- Sticky header və scroll-to-top düyməsi
- Mobil naviqasiya menyusu (hamburger)

## Texnologiyalar

- HTML5
- CSS3 (custom properties / dəyişənlər, Grid, Flexbox, media query-lər)
- Vanilla JavaScript (kitabxanasız)

## Necə işə salmaq olar

Repo-nu klonlayın və `index.html` faylını brauzerdə açın, əlavə quraşdırma tələb olunmur.

```bash
git clone https://github.com/alys27/novatech-landing-page.git
cd novatech-landing-page
```

Sonra `index.html`-i brauzerdə açın, və ya GitHub Pages üzərindən yuxarıdakı demo linkinə baxın.
